## Build

Before starting install `Node.js` and `yarn`.

### Step 1: Install
```sh
yarn install
```

### Step 2: Develop & Build

Once installed or cloned locally and all packages are installed you can begin developing your site.

```sh
# Run localhost for dev
yarn start

# Build site for prod
yarn build
```
