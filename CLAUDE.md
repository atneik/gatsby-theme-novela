# gatsby-theme-novela

## Node Version

This project requires **Node 12**. Running on Node 17+ breaks webpack 4 (OpenSSL `unsupported` error).

- `.nvmrc` is set to `12` — run `nvm use` when entering the project
- To auto-switch on `cd`, add the nvm chpwd hook to `~/.zshrc` (see README or nvm docs)

## Install

```bash
yarn install
```

## Dev

```bash
yarn start
```
